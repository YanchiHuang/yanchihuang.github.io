## 第一次使用git

安裝git之後，需要先做的設定

## 設定使用者與email

``` bash
git config --global user.name "yanchi.huang"
git config --global user.email "yanchi.huang@gmail.com"
```

## 查看git config

``` bash
git config --list
```
